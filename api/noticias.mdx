## Overview

The Noticias API allows you to manage news articles in the MTB Backend system. Each article includes a title, slug, main image, summary, content, publication date, author, and publication status.

<Note>
This endpoint supports draft and publish functionality. Use the `publishedAt` field to control publication status.
</Note>

---

## List All Noticias

<api-endpoint method="GET" path="/api/noticias" />

Retrieve a list of all news articles with optional filtering, sorting, and pagination.

### Query Parameters

<ParamField query="filters" type="object" optional>
  Filter articles by field values. Example: `filters[publicado][$eq]=true`
</ParamField>

<ParamField query="sort" type="string" optional>
  Sort articles by field(s). Example: `fecha:desc` for newest first
</ParamField>

<ParamField query="pagination[page]" type="number" optional default="1">
  Page number for pagination
</ParamField>

<ParamField query="pagination[pageSize]" type="number" optional default="25">
  Number of items per page
</ParamField>

<ParamField query="populate" type="string" optional>
  Populate relations. Use `*` for all relations or specify fields like `imagenPrincipal`
</ParamField>

### Response

<ResponseField name="data" type="array">
  Array of noticia objects
  
  <Expandable title="Noticia Object">
    <ResponseField name="id" type="number">
      Unique identifier for the article
    </ResponseField>
    
    <ResponseField name="attributes" type="object">
      <Expandable title="Attributes">
        <ResponseField name="titulo" type="string" required>
          Title of the news article
        </ResponseField>
        
        <ResponseField name="slug" type="string">
          URL-friendly slug generated from the title
        </ResponseField>
        
        <ResponseField name="imagenPrincipal" type="object">
          Main image media object
        </ResponseField>
        
        <ResponseField name="resumen" type="string" required>
          Brief summary of the article
        </ResponseField>
        
        <ResponseField name="contenido" type="array" required>
          Full article content in blocks format
        </ResponseField>
        
        <ResponseField name="fecha" type="date" required>
          Publication date of the article
        </ResponseField>
        
        <ResponseField name="autor" type="string">
          Name of the article author
        </ResponseField>
        
        <ResponseField name="publicado" type="boolean" default="true">
          Whether the article is published
        </ResponseField>
        
        <ResponseField name="publishedAt" type="datetime">
          Publication timestamp (null if draft)
        </ResponseField>
        
        <ResponseField name="createdAt" type="datetime">
          Creation timestamp
        </ResponseField>
        
        <ResponseField name="updatedAt" type="datetime">
          Last update timestamp
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata
  
  <Expandable title="Meta Object">
    <ResponseField name="pagination" type="object">
      <ResponseField name="page" type="number">
        Current page number
      </ResponseField>
      <ResponseField name="pageSize" type="number">
        Number of items per page
      </ResponseField>
      <ResponseField name="pageCount" type="number">
        Total number of pages
      </ResponseField>
      <ResponseField name="total" type="number">
        Total number of articles
      </ResponseField>
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/api/noticias?populate=*&sort=fecha:desc&filters[publicado][$eq]=true" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/noticias?populate=*&sort=fecha:desc&filters[publicado][$eq]=true', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/api/noticias',
    params={
        'populate': '*',
        'sort': 'fecha:desc',
        'filters[publicado][$eq]': 'true'
    },
    headers={'Content-Type': 'application/json'}
)
data = response.json()
```
</CodeGroup>

## Create a Noticia

<api-endpoint method="POST" path="/api/noticias" />

Create a new news article.

### Request Body

<ParamField body="data" type="object" required>
  <Expandable title="Noticia Data">
    <ParamField body="titulo" type="string" required>
      Title of the news article
    </ParamField>
    
    <ParamField body="slug" type="string" optional>
      URL-friendly slug (auto-generated from title if not provided)
    </ParamField>
    
    <ParamField body="imagenPrincipal" type="number" optional>
      ID of the uploaded main image media file
    </ParamField>
    
    <ParamField body="resumen" type="string" required>
      Brief summary of the article
    </ParamField>
    
    <ParamField body="contenido" type="array" required>
      Full article content in blocks format
    </ParamField>
    
    <ParamField body="fecha" type="date" required>
      Publication date (YYYY-MM-DD format)
    </ParamField>
    
    <ParamField body="autor" type="string" optional>
      Name of the article author
    </ParamField>
    
    <ParamField body="publicado" type="boolean" optional default="true">
      Whether the article is published
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="data" type="object">
  The created noticia object
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X POST "https://api.example.com/api/noticias" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "data": {
      "titulo": "Nueva Ruta de MTB Inaugurada",
      "resumen": "Se inaugura una nueva ruta de mountain bike en la región.",
      "contenido": [
        {
          "type": "paragraph",
          "children": [{"type": "text", "text": "La nueva ruta ofrece 15 km de pistas..."}]
        }
      ],
      "fecha": "2024-01-25",
      "autor": "María González",
      "publicado": true
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/noticias', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({
    data: {
      titulo: 'Nueva Ruta de MTB Inaugurada',
      resumen: 'Se inaugura una nueva ruta de mountain bike en la región.',
      contenido: [
        {
          type: 'paragraph',
          children: [{type: 'text', text: 'La nueva ruta ofrece 15 km de pistas...'}]
        }
      ],
      fecha: '2024-01-25',
      autor: 'María González',
      publicado: true
    }
  })
});
const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/api/noticias',
    headers={
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_TOKEN'
    },
    json={
        'data': {
            'titulo': 'Nueva Ruta de MTB Inaugurada',
            'resumen': 'Se inaugura una nueva ruta de mountain bike en la región.',
            'contenido': [
                {
                    'type': 'paragraph',
                    'children': [{'type': 'text', 'text': 'La nueva ruta ofrece 15 km de pistas...'}]
                }
            ],
            'fecha': '2024-01-25',
            'autor': 'María González',
            'publicado': True
        }
    }
)
data = response.json()
```
</CodeGroup>

## Update a Noticia

<api-endpoint method="PUT" path="/api/noticias/:id" />

Update an existing news article by its ID.

### Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the article to update
</ParamField>

### Request Body

<ParamField body="data" type="object" required>
  Noticia fields to update (same structure as Create)
</ParamField>

### Response

<ResponseField name="data" type="object">
  The updated noticia object
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X PUT "https://api.example.com/api/noticias/1" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "data": {
      "titulo": "Gran Evento de MTB 2024 - Actualizado",
      "publicado": false
    }
  }'
```
