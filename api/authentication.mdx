The MTB Backend API uses JWT (JSON Web Token) authentication powered by Strapi's `@strapi/plugin-users-permissions` plugin. This guide covers user registration, login, and how to use JWT tokens in your API requests.

## Authentication Flow

<Steps>
  <Step title="Register a new user">
    Create a new user account by providing username, email, and password
  </Step>
  <Step title="Login to get JWT token">
    Authenticate with your credentials to receive a JWT token
  </Step>
  <Step title="Use token in requests">
    Include the JWT token in the Authorization header for protected endpoints
  </Step>
</Steps>

## Register a New User

Create a new user account by sending a POST request to the `/api/auth/local/register` endpoint.

### Request

<ParamField path="username" type="string" required>
  Unique username for the new user
</ParamField>

<ParamField path="email" type="string" required>
  Valid email address for the user
</ParamField>

<ParamField path="password" type="string" required>
  Password for the user account (minimum 6 characters)
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST https://your-api-url.com/api/auth/local/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "johndoe",
    "email": "john.doe@example.com",
    "password": "SecurePass123!"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-api-url.com/api/auth/local/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    username: 'johndoe',
    email: 'john.doe@example.com',
    password: 'SecurePass123!'
  })
});

const data = await response.json();
console.log(data.jwt); // Your JWT token
```

```python Python
import requests

url = "https://your-api-url.com/api/auth/local/register"
payload = {
    "username": "johndoe",
    "email": "john.doe@example.com",
    "password": "SecurePass123!"
}

response = requests.post(url, json=payload)
data = response.json()
print(data['jwt'])  # Your JWT token
```

</CodeGroup>

### Response

<ResponseField name="jwt" type="string">
  JWT token for authentication
</ResponseField>

<ResponseField name="user" type="object">
  User information object
  
  <ResponseField name="id" type="number">
    User ID
  </ResponseField>
  
  <ResponseField name="username" type="string">
    Username
  </ResponseField>
  
  <ResponseField name="email" type="string">
    User email address
  </ResponseField>
  
  <ResponseField name="provider" type="string">
    Authentication provider (default: "local")
  </ResponseField>
  
  <ResponseField name="confirmed" type="boolean">
    Whether the user email is confirmed
  </ResponseField>
  
  <ResponseField name="blocked" type="boolean">
    Whether the user account is blocked
  </ResponseField>
  
  <ResponseField name="createdAt" type="string">
    Account creation timestamp
  </ResponseField>
  
  <ResponseField name="updatedAt" type="string">
    Last update timestamp
  </ResponseField>
</ResponseField>

## Login

Authenticate with existing credentials to receive a JWT token.

### Request

<ParamField path="identifier" type="string" required>
  Username or email address
</ParamField>

<ParamField path="password" type="string" required>
  User password
</ParamField>

### Example Request

<CodeGroup>

```bash cURL
curl -X POST https://your-api-url.com/api/auth/local \
  -H "Content-Type: application/json" \
  -d '{
    "identifier": "john.doe@example.com",
    "password": "SecurePass123!"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-api-url.com/api/auth/local', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    identifier: 'john.doe@example.com',
    password: 'SecurePass123!'
  })
});

const data = await response.json();
console.log(data.jwt); // Your JWT token
```

```python Python
import requests

url = "https://your-api-url.com/api/auth/local"
payload = {
    "identifier": "john.doe@example.com",
    "password": "SecurePass123!"
}

response = requests.post(url, json=payload)
data = response.json()
print(data['jwt'])  # Your JWT token
```

</CodeGroup>

## Using JWT Tokens in Requests

Once you have a JWT token, include it in the `Authorization` header of your requests to access protected endpoints.

### Header Format

```
Authorization: Bearer YOUR_JWT_TOKEN
```

### Example Authenticated Request

<CodeGroup>

```bash cURL
curl -X GET https://your-api-url.com/api/users/me \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

```javascript JavaScript
const response = await fetch('https://your-api-url.com/api/users/me', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...',
    'Content-Type': 'application/json'
  }
});

const user = await response.json();
```

```python Python
import requests

url = "https://your-api-url.com/api/users/me"
headers = {
    "Authorization": "Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
user = response.json()
```

</CodeGroup>

## Password Reset

Request a password reset email for a user account.

### Request Password Reset

<CodeGroup>

```bash cURL
curl -X POST https://your-api-url.com/api/auth/forgot-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john.doe@example.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-api-url.com/api/auth/forgot-password', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'john.doe@example.com'
  })
});

const data = await response.json();
```

</CodeGroup>

### Reset Password with Code

After receiving the reset code via email, use it to set a new password:

<CodeGroup>

```bash cURL
curl -X POST https://your-api-url.com/api/auth/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "code": "RESET_CODE_FROM_EMAIL",
    "password": "NewSecurePass123!",
    "passwordConfirmation": "NewSecurePass123!"
  }'
```

```javascript JavaScript
const response = await fetch('https://your-api-url.com/api/auth/reset-password', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    code: 'RESET_CODE_FROM_EMAIL',
    password: 'NewSecurePass123!',
    passwordConfirmation: 'NewSecurePass123!'
  })
});

const data = await response.json();
```

</CodeGroup>

## Error Responses

Authentication endpoints may return the following error responses:

### 400 Bad Request

```json
{
  "error": {
    "status": 400,
    "name": "ValidationError",
    "message": "Email or Username are already taken",
    "details": {}
  }
}
```

### 401 Unauthorized

```json
{
  "error": {
    "status": 401,
    "name": "UnauthorizedError",
    "message": "Invalid identifier or password",
    "details": {}
  }
}
```

### 403 Forbidden

```json
{
  "error": {
    "status": 403,
    "name": "ForbiddenError",
    "message": "Your account has been blocked by an administrator",
    "details": {}
  }
}
```

## Best Practices

<Warning>
  **Security Considerations**
  
  - Always use HTTPS in production to protect JWT tokens in transit
  - Store JWT tokens securely (e.g., httpOnly cookies, secure storage)
  - Never expose tokens in URLs or logs
  - Implement token refresh mechanisms for long-lived sessions
  - Set appropriate token expiration times
</Warning>

### Token Storage

- **Web Applications**: Use httpOnly cookies or secure browser storage (localStorage with proper XSS protection)
- **Mobile Applications**: Use secure storage mechanisms (iOS Keychain, Android Keystore)
- **Server-to-Server**: Use environment variables or secure secret management systems

### Token Expiration

JWT tokens issued by Strapi have a default expiration time. When a token expires, users must re-authenticate by calling the login endpoint again.

To check if a token is expired, you'll receive a 401 Unauthorized response:

```json
{
  "error": {
    "status": 401,
    "name": "UnauthorizedError",
    "message": "Invalid token: Token expired",
    "details": {}
  }
}
```
