## Overview

The Inscritos API allows you to manage registered participants in the MTB Backend system. Each inscrito represents a confirmed participant with their details, age, category, and registration type.

<Note>
This endpoint supports draft and publish functionality. Use the `publishedAt` field to control publication status.
</Note>

---

## List All Inscritos

<api-endpoint method="GET" path="/api/inscritos" />

Retrieve a list of all registered participants with optional filtering, sorting, and pagination.

### Query Parameters

<ParamField query="filters" type="object" optional>
  Filter participants by field values. Example: `filters[categoria][$eq]=Elite`
</ParamField>

<ParamField query="sort" type="string" optional>
  Sort participants by field(s). Example: `nombreCompleto:asc`
</ParamField>

<ParamField query="pagination[page]" type="number" optional default="1">
  Page number for pagination
</ParamField>

<ParamField query="pagination[pageSize]" type="number" optional default="25">
  Number of items per page
</ParamField>

### Response

<ResponseField name="data" type="array">
  Array of inscrito objects
  
  <Expandable title="Inscrito Object">
    <ResponseField name="id" type="number">
      Unique identifier for the participant
    </ResponseField>
    
    <ResponseField name="attributes" type="object">
      <Expandable title="Attributes">
        <ResponseField name="nombreCompleto" type="string" required>
          Full name of the participant
        </ResponseField>
        
        <ResponseField name="rut" type="string" required>
          Chilean national ID (RUT) of the participant
        </ResponseField>
        
        <ResponseField name="edad" type="number" required>
          Age of the participant (between 3 and 99)
        </ResponseField>
        
        <ResponseField name="categoria" type="string" required>
          Competition category: `Infantil`, `Experto`, `Enduro`, `Elite`, `Master A`, or `Master B`
        </ResponseField>
        
        <ResponseField name="tipo" type="string" required>
          Registration type: `Open` or `Federado`
        </ResponseField>
        
        <ResponseField name="publishedAt" type="datetime">
          Publication timestamp (null if draft)
        </ResponseField>
        
        <ResponseField name="createdAt" type="datetime">
          Creation timestamp
        </ResponseField>
        
        <ResponseField name="updatedAt" type="datetime">
          Last update timestamp
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Pagination metadata
  
  <Expandable title="Meta Object">
    <ResponseField name="pagination" type="object">
      <ResponseField name="page" type="number">
        Current page number
      </ResponseField>
      <ResponseField name="pageSize" type="number">
        Number of items per page
      </ResponseField>
      <ResponseField name="pageCount" type="number">
        Total number of pages
      </ResponseField>
      <ResponseField name="total" type="number">
        Total number of participants
      </ResponseField>
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/api/inscritos?filters[categoria][$eq]=Elite&sort=nombreCompleto:asc" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/inscritos?filters[categoria][$eq]=Elite&sort=nombreCompleto:asc', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/api/inscritos',
    params={
        'filters[categoria][$eq]': 'Elite',
        'sort': 'nombreCompleto:asc'
    },
    headers={'Content-Type': 'application/json'}
)
data = response.json()
```
</CodeGroup>

  "data": [

      "id": 1,

        "nombreCompleto": "Diego Fernández",

## Create an Inscrito

<api-endpoint method="POST" path="/api/inscritos" />

Create a new registered participant entry.

### Request Body

<ParamField body="data" type="object" required>
  <Expandable title="Inscrito Data">
    <ParamField body="nombreCompleto" type="string" required>
      Full name of the participant
    </ParamField>
    
    <ParamField body="rut" type="string" required>
      Chilean national ID (RUT) of the participant
    </ParamField>
    
    <ParamField body="edad" type="number" required>
      Age of the participant (must be between 3 and 99)
    </ParamField>
    
    <ParamField body="categoria" type="string" required>
      Competition category. Must be one of: `Infantil`, `Experto`, `Enduro`, `Elite`, `Master A`, `Master B`
    </ParamField>
    
    <ParamField body="tipo" type="string" required>
      Registration type. Must be one of: `Open`, `Federado`
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="data" type="object">
  The created inscrito object
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X POST "https://api.example.com/api/inscritos" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "data": {
      "nombreCompleto": "Laura Rojas",
      "rut": "55667788-9",
      "edad": 27,
      "categoria": "Experto",
      "tipo": "Open"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/inscritos', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({
    data: {
      nombreCompleto: 'Laura Rojas',
      rut: '55667788-9',
      edad: 27,
      categoria: 'Experto',
      tipo: 'Open'
    }
  })
});
const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.example.com/api/inscritos',
    headers={
        'Content-Type': 'application/json',
        'Authorization': 'Bearer YOUR_TOKEN'
    },
    json={
        'data': {
            'nombreCompleto': 'Laura Rojas',
            'rut': '55667788-9',
            'edad': 27,
            'categoria': 'Experto',
            'tipo': 'Open'
        }
    }
)
data = response.json()
```
</CodeGroup>

### Example Response

## Categories and Types

### Available Categories

The `categoria` field must be one of the following values:

- **Infantil**: Children's category
- **Experto**: Expert level riders
- **Enduro**: Enduro discipline
- **Elite**: Elite professional level
- **Master A**: Master category A
- **Master B**: Master category B

### Available Types

The `tipo` field must be one of the following values:

- **Open**: Open registration for all participants
- **Federado**: Federated/official club members only

### Age Validation

The `edad` field must be between **3** and **99** years old.
